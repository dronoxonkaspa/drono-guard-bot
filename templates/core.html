Core Components

The Dronox Core stack is engineered for performance, resilience, and modularity — integrating key systems that power the Kaspa ecosystem.

Validator Nodes

Distributed verification endpoints ensuring consensus integrity across all Kaspa-connected layers.

Kaspa Link

Bridging systems for cross-layer communication, enabling data flow and liquidity between protocols.

API Gateway

Unified access for developers — real-time endpoints for analytics, transactions, and interoperability.

Node Relay

High-speed relay framework connecting validators, miners, and dApps under the Dronox Core architecture.
